Common Files

This page lists links to the files of common interest. There are several ways to download a file:
  • Click on the link
  • Right click on the link and select "Save Link As..."
  • Copy the link and then, on the computer where you want to use the file, run:
    • curl -LO https://copied-link
      • or
    • wget https://copied-link
