FCCAnalyses

Common analysis framework for the Future Circular Collider

About

The framework is used for the majority of the FCC related analyses, non definitive list can be found at FCCee Physics Performance. Individual analyses are designed to use the ROOT RDataFrame as the main event processor and the analyses input files contain physics objects in EDM4hep format.
