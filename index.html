About

The FCCAnalyses framework is used for the majority of the FCC related analyses, non definitive list of the analyses can be found at FCCee Physics Performance website.

Individual analyses are designed to use the ROOT RDataFrame as the main event processor and the analyses input files contain physics objects in EDM4hep format.

Using ROOT dataframe allows to use modern, high-level interface and very quick processing time as it natively supports multithreading.

Obtaining

The FCCAnalyses is distributed in the Key4hep stack: 
source /cvmfs/sw.hsf.org/key4hep/setup.sh
fccanalysis run analysis_script.py
